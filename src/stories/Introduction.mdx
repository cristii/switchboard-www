# Switchboard AI Systems — Design System

The brand system for **Switchboard AI Systems**, the practice of Cristi Șatcovschi,
who builds **AI chatbots and workflow automation** (n8n, Trigger.dev and similar)
for small businesses — assistants that answer visitor questions, qualify leads, and
book calls.

The visual identity is a warm **"paper & ink"** aesthetic: recycled-paper backgrounds
with a faint grid and grain, deep pine-green ink, a single burnt-orange accent,
hand-drawn underlines, and **hard, blur-free offset shadows**. It reads handmade and
confident — a craftsperson's site, not a SaaS template.

## How to use this library

- **Foundations** documents the raw language — Colors, Typography, Spacing & Layout,
  Shadows, and the bespoke Icon set.
- **Core / Surfaces / Data / Chat** are the React components. Each story has a **Docs**
  tab (props + usage) and **Controls** to try variants live.
- In the app, import everything from `src/components/ui` — see `AGENTS.md`. If a needed
  component or variant doesn't exist yet, **add it here first** (with a story), then
  consume it.

## The rules that define the brand

- **Hard, zero-blur offset shadows.** Solid ink/orange blocks — never soft glows. The
  most important and most easily-broken rule.
- **One accent.** Burnt orange `#B45309` for CTAs, links, numbers, and the hand-drawn
  underline. No gradients as identity.
- **Three fonts, three jobs.** Bricolage Grotesque (headings/UI), Inter (body), Caveat
  (hand marginalia + the ↳ glyph). Self-hosted, no CDN.
- **Bespoke line icons only.** No icon font, no third-party library.
- **No emoji, ever.** Personality comes from the Caveat hand, the ↳ glyph, and the
  hand-drawn underline.

## Voice

First-person singular, direct, anti-fluff, warm. Sells outcomes (leads, booked calls),
never features for their own sake. Honest numbers only — the brand never invents fake
statistics or names specific clients.
