Trigger.dev Webhooks Node.js

Catching Trigger.dev webhook timeouts gracefully

Wrap the outbound call in a bounded task so a slow upstream API retries instead of failing the whole run.

src/trigger/webhook.ts 
client.defineJob({
  id: "webhook-timeout-handler",
  run: async (payload, io) => {
    await io.runTask("fetch", { timeoutInMs: 8000 },
      () => fetch(payload.url)); // auto-retries
  },
});
