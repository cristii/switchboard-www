A rotated, taped paper note set in the Caveat hand — for playful asides like a tech-stack list. Use one per view, max; overuse cheapens it.

```jsx
<StickyNote title="Tech stack">
  <ul style={{ listStyle: "none" }}>
    <li>– n8n</li><li>– OpenAI / GPT</li><li>– Node.js / Nest</li>
  </ul>
</StickyNote>
```

`rotate` tunes the tilt (default -1.4°). The orange "tape" tab is drawn automatically.
